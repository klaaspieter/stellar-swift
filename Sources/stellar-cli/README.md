## Usage

```
Stellar

Usage:
  stellar accounts <id> [--network=NETWORK]
  stellar balance <id> [--network=NETWORK]
  stellar -h|--help

Options:
  --help                       Show this screen.
  --network=[main|test|URL]    The network to use. “main”, “test” or a URL for a non-default network. [Default: test]
```
